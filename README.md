# ydict.js

Yet another ydict implmentation in node.js

## Installation
- Install from repository
```shell
[sudo] npm install -g
```

- Install from NPM
```shell
[sudo] npm install -g ydict.js
```

## Usage

```shell
ydict.js <word or phrase>
```

## Playing Sound

Add a json file named `.ydict.json` in home dir and specify the player command.

```json
{
    "playerCmd": "mplayer"
}
```
